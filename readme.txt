How to setup:
=============

1. Download msysgit from http://code.google.com/p/msysgit/.
2. Create an Account in http://github.com if you want to participate with coding.
3. Follow the instructions in GitHub how to setup and configure your account.
4. Select a location in your hard drive where you want to download the code.
5. Right click and select "Git Bash Here" it will open the console.
6. Now type git clone git://github.com/kazimanzurrashid/LetsCodeWithMe.git, this will copy the repository in your local hard drive.
7. Now Run the source\InstallPackages.bat in the console, it will install the required packages from NuGet.
8. Now Open the Solution in the Visual Studio.
